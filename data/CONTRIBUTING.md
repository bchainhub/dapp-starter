# Contributing

Thank you for your interest in contributing.

## How to contribute

1. **Open an issue** – Report bugs, suggest features, or ask questions in the repository’s **Issues** tab.
2. **Fork and clone** – Fork the repo and clone it locally.
3. **Create a branch** – Use a short, descriptive branch name (e.g. `fix/typo-readme`, `feat/add-xyz`).
4. **Make changes** – Keep commits focused and messages clear.
5. **Push and open a PR** – Push your branch and open a Pull Request against the default branch. Describe what changed and why.

## Code and style

- Follow the existing style and structure of the project.
- Ensure any new code is appropriately tested or documented as needed.
